# SubAgents

| Agent | Purpose | When to Use |
|-------|---------|-------------|
| backend-api-developer | REST API 개발, 데이터베이스 운영, 인증/권한, 비즈니스 로직 | 서버 사이드 로직 구현 시 |
| frontend-ui-developer | UI 컴포넌트 구현, 반응형 레이아웃, 디자인 스펙 적용 | 프론트엔드 개발 시 |
| ui-ux-designer | 인터페이스 분석, 사용자 플로우 평가, 와이어프레임 생성 | UI/UX 디자인 작업 시 |
| web-app-tester | Playwright 테스트, 성능 모니터링, UI/UX 검증 | E2E 테스트, 회귀 테스트 시 |
| code-debugger | 버그 식별/분석/해결, 에러 분석, 실행 경로 추적 | 디버깅, 오류 수정 시 |
| technical-documentation-writer | 기술 문서 작성, .serena/memories 관리 | 문서화, 메모리 업데이트 시 |
| task-planner | 작업 계획 수립 및 문서화 | 복잡한 작업 계획 필요 시 |
| task-router | 작업 분석 및 적합한 에이전트 추천 | 작업 할당 결정 시 |
| data-scientist | EDA, 통계 모델링, ML 실험, 가설 검정 | 데이터 분석 전반 |
| data-cleaning-specialist | 결측값 처리, 이상치 탐지, 데이터 검증, 품질 평가 | 데이터 전처리 시 |
| data-visualization-specialist | matplotlib/seaborn/plotly 시각화, 분포도, 히트맵 | 데이터 시각화 시 |
| feature-engineering-specialist | 특성 생성, 인코딩, 스케일링, 차원 축소 | 피처 엔지니어링 시 |
| ml-modeling-specialist | 모델 선택/훈련, 하이퍼파라미터 튜닝, AutoML | ML 모델 개발 시 |
| model-evaluation-specialist | 성능 지표, 혼동 행렬, ROC, 모델 비교 | 모델 평가/비교 시 |
